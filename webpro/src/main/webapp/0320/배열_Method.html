indexOf(item[, start]) 아이템의 위치 찾기
lastIndexOf(item[, start]) 역순으로 위치찾기



==================================

push(a, b, c, ...) : 배열의 마지막에 요소를 추가
pop() : 배열의 마지막 요소를 제거하고 반환



==================================

arrNum = [10, 23, 100, 115, 200, 5]
sort() : 요소를 정렬



==================================

slice(start, end) : start ~ end-1 까지 추출하여 새로운 배열로 생성



==================================

splice(index, [n], [a, b, c, ...]) index부터 n개를 제거
[a, b, c, ...]요소 삽입
n = 0일 때, 삭제없이 진행 / a, b, c요소를 생략하면 삽입 없음



==================================

a.concat(b, c) : a 배열에 b, c 배열을 추가



==================================

join(구분자) : 배열을 하나의 문자열로 변환
